# primeiro-projeto-react

Protótipo em React desenvolvido para exploração de recursos fundamentais da biblioteca e realização de testes de performance.

---

## Objetivos do Projeto
Este repositório serve como um ambiente de sandbox para:
* Validar o ciclo de vida de componentes.
* Testar a eficiência de renderização (Performance).
* Praticar a integração de hooks básicos (useState, useEffect).

##  Como executar o projeto

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

1. **Clone o repositório:**
   ```bash
   git clone [https://github.com/seu-usuario/primeiro-projeto-react.git](https://github.com/seu-usuario/primeiro-projeto-react.git)
Entre na pasta do projeto:

Bash
cd my-app
Instale as dependências:

Bash
npm install
Inicie o servidor de desenvolvimento:

Bash
npm run dev
[!TIP]
O comando acima abrirá o projeto em uma porta local (geralmente http://localhost:5173/ se estiver usando Vite).

Tecnologias Utilizadas
React (Biblioteca principal)

Vite (Build tool rápida)

JavaScript/JSX
