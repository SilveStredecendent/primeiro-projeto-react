# primeiro-projeto-react

Protótipo em React desenvolvido para exploração de recursos fundamentais da biblioteca e realização de testes de performance.

---

## Objetivos do Projeto
Este repositório serve como um ambiente de **sandbox** (espaço de testes) para:

* **Validar** o ciclo de vida de componentes funcionais.
* **Testar** a eficiência de renderização e otimização de performance.
* **Praticar** a integração de hooks básicos, especificamente `useState` e `useEffect`.

---

## Como Executar o Projeto
Você pode rodar este projeto diretamente na nuvem (via GitHub Codespaces) ou em sua máquina local.

### Opção 1: No GitHub Codespaces
Se você estiver utilizando o ambiente do GitHub:

1. No terminal do seu Codespace, certifique-se de estar na pasta raiz do projeto.
2. **Instale as dependências necessárias:**
   ```bash
   npm install

```

3. **Inicie o servidor de desenvolvimento:**
```bash
npm run dev

```


4. Um balão aparecerá no canto inferior direito; clique em **"Abrir no Navegador"**.

---

### Opção 2: Execução Local

Para rodar em sua própria máquina, você precisará do [Node.js](https://nodejs.org/) instalado.

1. **Clone o repositório:**
```bash
git clone [https://github.com/seu-usuario/primeiro-projeto-react.git](https://github.com/seu-usuario/primeiro-projeto-react.git)

```


2. **Entre na pasta do projeto:**
```bash
cd primeiro-projeto-react

```


3. **Instale as dependências:**
```bash
npm install

```


4. **Inicie o projeto:**
```bash
npm run dev

```


5. O terminal indicará o endereço local (ex: `http://localhost:5173`). Abra-o em seu navegador.
