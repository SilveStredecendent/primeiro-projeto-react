## primeiro-projeto-react
Protótipo em React desenvolvido para exploração de recursos fundamentais da biblioteca e realização de testes de performance.

## Objetivos do Projeto
Este repositório serve como um ambiente de sandbox (espaço de testes) para:

Validar o ciclo de vida de componentes funcionais.

Testar a eficiência de renderização e otimização de performance.

Praticar a integração de hooks básicos, especificamente useState e useEffect.

## Como Executar o Projeto
Você pode rodar este projeto diretamente na nuvem (via GitHub Codespaces) ou em sua máquina local.

## Opção 1: No GitHub Codespaces
Se você estiver utilizando o ambiente do GitHub:

No terminal do seu Codespace, certifique-se de estar na pasta raiz do projeto.

Instale as dependências necessárias:

Bash
npm install
Inicie o servidor de desenvolvimento:

Bash
npm run dev
Um balão aparecerá no canto inferior direito; clique em "Abrir no Navegador".

## Opção 2: Execução Local
Para rodar em sua própria máquina, você precisará do Node.js instalado.

Clone o repositório:

Bash
git clone https://github.com/seu-usuario/primeiro-projeto-react.git
Entre na pasta do projeto:

Bash
cd primeiro-projeto-react
Instale as dependências:

Bash
npm install
Inicie o projeto:

Bash
npm run dev
O terminal indicará o endereço local (ex: http://localhost:5173). Abra-o em seu navegador.

## Tecnologias Utilizadas
React (Biblioteca UI)

Vite (Ferramenta de build e servidor dev)

JavaScript / JSX

## Testes de Performance
Como o foco deste projeto é a performance, recomendo utilizar a extensão React DevTools no navegador para monitorar as renderizações.

[!NOTE]
Para resultados mais precisos em testes de performance, prefira a Execução Local, pois evita latências causadas pela conexão de rede do Codespaces.
